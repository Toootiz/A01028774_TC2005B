CSS Cheat Sheet

border

La propiedad border permite definir el borde de un elemento.

Ejemplo de border

border-left, border-right, border-top, border-bottom

Estas propiedades permiten definir el borde en un lado específico del elemento.

Ejemplo de border-left, border-right, border-top, border-bottom

border-radius

La propiedad border-radius permite redondear las esquinas de un borde.

Ejemplo de border-radius

background-image

La propiedad background-image define una imagen de fondo para un elemento.

Ejemplo de background-image

color

La propiedad color define el color del texto.

Ejemplo de color

font

La propiedad font permite definir el estilo, tamaño y familia de fuente en una sola declaración.

Ejemplo de font

font-family

La propiedad font-family define la familia de fuentes para el texto.

Ejemplo de font-family

font-style, font-weight

La propiedad font-style define el estilo de la fuente (normal, cursiva) y font-weight define el grosor de la fuente.

Ejemplo de font-style y font-weight

width, height

Las propiedades width y height definen el ancho y la altura de un elemento.

Ejemplo de width y height

list-style

La propiedad list-style define el estilo de las listas.

Ejemplo de list-style

margin

La propiedad margin define el espacio exterior de un elemento.

Ejemplo de margin

padding

La propiedad padding define el espacio interior de un elemento.

Ejemplo de padding

overflow

La propiedad overflow define cómo se manejará el contenido desbordante.

Ejemplo de overflow

display

La propiedad display define cómo se muestra un elemento en el flujo del documento.

Ejemplo de display

position

La propiedad position define el método de posicionamiento de un elemento.

Ejemplo de position

filter

La propiedad filter aplica efectos gráficos a un elemento.

Ejemplo de filter
